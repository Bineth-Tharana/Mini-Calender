# 📅 Mini Calendar

A simple and responsive **Mini Calendar** built using **HTML, CSS and JavaScript**. This project displays the current day, date, month and year with a clean and modern user interface. It is a lightweight front-end project suitable for beginners learning JavaScript date manipulation.

---

## 🚀 Live Demo

🔗 https://bineth-tharana.github.io/Mini-Calender/

---

## ✨ Features

- 📅 Displays the current date automatically
- 📆 Shows the current day, month and year
- ⚡ Updates dynamically using JavaScript
- 🎨 Clean and modern user interface
- 📱 Responsive design for different screen sizes
- 💻 Lightweight with no external libraries

---

## ⚙️ How It Works

1. JavaScript retrieves the current date using the `Date()` object.
2. The day, date, month and year are extracted.
3. These values are inserted into the HTML elements dynamically.
4. The calendar automatically reflects the user's current date whenever the page is loaded.
